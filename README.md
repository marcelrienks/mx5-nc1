# MX5-NC1

A comprehensive repository for Mazda MX-5 NC1 PRHT documentation, 3D CAD models, technical manuals, and DIY maintenance references.

## Overview

This repository contains personal reference materials, CAD designs, and technical documentation for a Mazda MX-5 NC1 (2009-2015 third-generation Miata). It includes wheel alignment specifications, custom 3D-printed parts models, factory service manuals, and DIY adjustment guides.

---

## Contents Overview

### Wheel Alignment Documentation

**File:** `MX5 NC1 Home Alignment.md`

A comprehensive guide covering:
- **Alignment presets** for different driving styles (Flyin Miata, Fast Road, and a middle-ground specification)
- **DIY string box method** with step-by-step instructions for at-home alignment measurement and adjustment
- **mm-to-degree conversion formulas** for toe and camber
- **Caster measurement techniques** using a 15° steering sweep method
- **Factory specifications** including wheelbase, track width, bolt pattern, and common fitment details for the NC1 PRHT
- **Practical troubleshooting tips** and measurement best practices

**Highlights:**
- Targets optimized for both street and spirited driving
- Methods using affordable DIY tools (string, plumb line, ruler, PVC pipe)
- Detailed math and formulas for converting between angle measurements and practical mm references
- Repeatability-focused procedures

### 3D CAD Models

**Location:** `Models/`

Fusion 360 (.f3d) CAD files for custom 3D-printed or machined parts:
- **Comfort & convenience:** door pull plugs, shift cup, phone mount, cubby organizer
- **Safety:** seat belt guides (multiple revisions), battery cover
- **Aesthetics:** grille caps, aerial adapter

All models are revision-tracked (R1, R2, etc.) indicating iterative refinement.

### Service Manuals

**Location:** `Manuals/`

Factory service documentation:
- **PRHT-specific:** Power Retractable Hard Top lubrication guide
- **Electrical:** Wiring diagrams and electrical system info
- **Powertrain:** Engine workshop manual covering 2.0L and 2.3L engines (used in MX-5, Mazda3, Mazda5, Mazda6, and other models)
- **Complete FSM:** 2011 MX-5 Factory Service Manual
- **Chassis:** Suspension, steering, and drivetrain reference documents

## Using This Repository

### For Wheel Alignment
1. Read the **Quickstart** section in `MX5 NC1 Home Alignment.md` for mm targets
2. Follow the **DIY String Box Method** for measurement setup and adjustment
3. Use the **Calculations & Examples** section to convert between angle specs and practical mm values
4. Reference **Factory / Baseline Dimensions** for chassis and wheel fitment details

### For CAD Models
- Open `.f3d` files in **Fusion 360** (free with education/personal use license)
- Modify, export to STL/STEP, or 3D print as needed
- Models are iterative; check the revision number for the latest version

### For Service Information
- Reference **Manuals/** for factory specifications and procedures
- Use the **Engine Workshop Manual** for powertrain diagnostics and service
- Consult **Wiring Diagram** and electrical guides for electrical troubleshooting
- Review **PRHT Lubrication Guide** if your vehicle has a power retractable hardtop

## Notes

- All alignment specifications and DIY procedures are personal reference targets and methods

- Always consult factory service manuals for torque specs, fluid types, and safety procedures
